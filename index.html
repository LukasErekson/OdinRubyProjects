Ruby Projects

Ruby exercise solutions and projects from The Odin Project

This is a compilation of all the exercises and projects I've done as part of The Odin Project's Ruby Basics course. The repository contains all my solutions to the regular ruby exercises, but I'm particularly proud of my work on the projects.

The pages on this website will primarily focus on the important concepts I learned during the projects rather than the projects themselves. If you're interested in understanding the initial requirements for the project, specific links will be provided. on each project page.

I tried to go above and beyond for each of the projects, so I hope you can take the time to see them in action. I tried to learn and implement a new best-practice in each of my successive projects. For instance, several projects come with their own unit tests using RSpec while others also have their own RDoc documentation.
